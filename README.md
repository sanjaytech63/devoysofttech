# DevoySoftTech SaaS Landing

A production-ready modern SaaS landing page using Next.js App Router, TypeScript (strict), Tailwind CSS, next-themes, next/image, and next/font.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Features

- feature-based modular structure (`components`, `sections`, `hooks`, `lib`, `config`)
- responsive light/dark mode with `next-themes`
- optimized fonts with `next/font`
- animated scroll reveal and hover transitions
- semantic accessibility-focused components
- SEO metadata, openGraph, twitter cards
- dark/light theme variables in `globals.css`
- 404 page and error boundary
